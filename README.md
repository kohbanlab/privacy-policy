# KohbanLab – Privacy Policy Center

This repository serves as the official privacy policy center for all applications developed and published by **KohbanLab**.

All policies available here are publicly accessible and hosted via GitHub Pages to ensure full transparency, accessibility, and compliance with applicable regulations, including the **Google Play Developer Policy** and the **General Data Protection Regulation (GDPR)**.

## Commitment to Privacy

KohbanLab is committed to protecting user privacy and ensuring responsible data handling practices.

Our applications are designed with a **privacy-first approach**, prioritizing:

- Minimal data collection  
- Local data storage whenever possible  
- No unnecessary data sharing  
- Clear and transparent user information  

---

## Purpose of This Repository

This repository aims to:

- Provide centralized access to all privacy policies  
- Ensure compliance with platform requirements (Google Play Store)  
- Clearly inform users about how their data is handled  
- Maintain a consistent and professional privacy framework across all applications  

---

## Policy Scope

Each privacy policy includes:

- Information about data collection (if applicable)  
- Description of how data is used  
- Details on data storage (local or remote)  
- Disclosure of any data sharing practices  
- Security measures  
- User rights under GDPR  
- Policy update procedures
